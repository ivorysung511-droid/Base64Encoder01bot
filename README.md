# Base64 Encoder/Decoder Bot

A Telegram bot that encodes and decodes Base64 for text and files.

## Features
- Encode text to Base64
- Decode Base64 to text
- Encode files (any type) to Base64
- Decode Base64 files back to original file
- Preview for Base64-encoded images

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Create `.env` file with your bot token
3. Run: `python bot.py`

## Deployment on Railway
1. Push code to GitHub
2. Connect repository to Railway
3. Add `BOT_TOKEN` as an environment variable
4. Deploy!
